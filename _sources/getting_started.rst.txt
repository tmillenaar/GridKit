Getting Started
================

What is GridKit?
----------------

GridKit is an open source Python library that provides a high-level API to work with regular grids.

Some of the core features are:

 - tessellation of an infinite surface
 - working with geospatial data
 - bridging the gap between raster and vector data
     - aggregation
     - sampling
     - interpolation
